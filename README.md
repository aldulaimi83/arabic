This is the Arabic learning website for kids and families learning Arabic abroad.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Static export ready for GitHub Pages or any static host

## Local development

First, run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build for production

```bash
npm run build
```

This repo is configured with static export in `next.config.ts`, so the build output can be deployed without Vercel.

## Deployment

You can deploy this site to:

- GitHub Pages
- Cloudflare Pages
- Netlify
- any static file host

If you use GitHub Pages, publish the exported `out/` directory through your Pages workflow or branch setup.
