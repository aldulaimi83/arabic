## 🆘 Troubleshooting

### "ChromaMaze won't load"
- Check `chromamaze.html` and `chromamaze.js` are in root
- Verify script paths are correct
- Clear browser cache (Ctrl+Shift+Del)

### "Leaderboard not appearing"
- Make sure `leaderboard.js` is loaded in index.html
- Check browser console for errors (F12)
- Verify localStorage isn't disabled

### "Scores not saving"
- Check if localStorage is enabled
- Try in private window (might have limits)
- Check browser allows local storage

### "Mobile responsiveness issues"
- Viewport meta tag should be present (it is)
- Test with Chrome DevTools (F12 → Toggle device)
- Check CSS media queries are working
